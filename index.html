How to Add More Books

Open this file and edit the BOOK_DATA array in the script below. Each book is one object. The homepage builds itself (category chips, counts, grids, and filters).

1) Add a Category

Add a new object like:
{ id: "romance", name: "Romance", badge: "Hot", badgeStyle: "o", books: [] }

2) Add a Book

Add a book into books: []:
{ title, author, year, pages, format, tags, detailsUrl, downloadUrl }

3) Details Page (Optional)

If you create per-book pages later, set detailsUrl to something like:
/books/atomic-habits.html

4) Safe Linking Tip

If a download is external, keep rel="noopener noreferrer" (already included).
